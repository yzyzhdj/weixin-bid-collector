# 查标讯 - 微信小程序

一个基于微信原生小程序开发的招标信息查询平台，提供招标公告浏览、筛选、搜索等功能。

## 功能特性

### 📱 页面功能

| 页面 | 功能描述 |
|------|----------|
| **首页** | 标讯列表展示（默认5条）、下拉刷新、省市两级联动筛选、公告类型筛选、招标方式筛选 |
| **商机** | 关键词搜索、热门标签、功能入口导航、招标统计数据 |
| **详情** | 标讯详情、关联公告、收藏分享功能 |
| **个人中心** | 用户信息、会员卡片、订单管理、优惠券、功能菜单 |
| **更多** | 标书模版、业务拓展、数据服务等功能入口 |

### 🔍 核心功能

- **智能筛选**：支持地区（省→市联动）、公告类型、招标方式多维筛选
- **下拉刷新**：实时获取最新标讯数据
- **上拉加载**：无限滚动加载更多内容
- **搜索功能**：支持关键词搜索标讯
- **收藏分享**：收藏感兴趣的标讯，分享给好友

## 技术架构

### 项目结构

```
├── app.json          # 小程序配置（页面路由、TabBar）
├── app.js            # 小程序入口文件
├── app.wxss          # 全局样式
├── project.config.json # 开发者工具配置
├── sitemap.json      # 站点地图
├── utils/
│   ├── config.js     # API配置（API_KEY）
│   ├── api.js        # API请求封装
│   └── city-data.js  # 省市联动数据
├── images/           # 图标资源
└── pages/
    ├── subscription/ # 首页（标讯列表）
    ├── index/        # 商机页（搜索）
    ├── list/         # 搜索结果页
    ├── detail/       # 标讯详情页
    ├── profile/      # 个人中心
    └── more/         # 更多功能
```

### 技术栈

- **框架**: 微信原生小程序
- **语言**: JavaScript ES6+
- **样式**: WXSS（微信小程序样式）
- **API**: RESTful API

## 快速开始

### 环境要求

- 微信开发者工具
- Node.js（可选，用于开发辅助）

### 安装步骤

1. **克隆仓库**

```bash
git clone https://github.com/yzyzhdj/weixin-bid-collector.git
```

2. **打开项目**

在微信开发者工具中选择"导入项目"，选择项目目录。

3. **配置说明**

API Key 已配置在 `utils/config.js` 中：

```javascript
const API_KEY = 'bid_8560c3dece3520c9a670a71a242e2343be6d50a18ecfe4b6';
```

4. **运行项目**

在微信开发者工具中点击"编译"按钮即可预览。

## API 接口

### 接口列表

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/bids` | GET | 获取标讯列表 |
| `/api/v1/bids/:id` | GET | 获取标讯详情 |
| `/api/v1/bids/search` | GET | 搜索标讯 |
| `/api/v1/stats` | GET | 获取统计数据 |
| `/api/v1/filters` | GET | 获取筛选选项 |

### 请求配置

- **基础URL**: `https://biaoxun.pandaorder.cn/api/v1`
- **认证方式**: Bearer Token（API_KEY）
- **超时时间**: 30秒

## 页面说明

### 首页 (subscription)

- 顶部标签筛选：全部、交易公告、交易结果、公开招标、竞争性磋商、竞争性谈判、询价
- 筛选栏：地区（省→市联动）、公告类型、招标方式
- 标讯列表：标题、类型标签、来源、地区、预算、日期

### 商机页 (index)

- 搜索栏：输入关键词搜索
- 热门标签：快速搜索热门关键词
- 功能入口：招标查询、中标查询、采购意向、拟建查询、供应商库
- 招标统计：今日更新、30天累计数据

### 详情页 (detail)

- 公告标题与基本信息
- 详细内容展示
- 关联招标/中标公告
- 收藏与分享按钮

### 个人中心 (profile)

- 用户信息展示
- 快捷操作：历史浏览、收藏项目、我的笔记
- 会员信息卡片
- 订单管理、优惠券
- 功能菜单列表

### 更多页 (more)

- 标书模版、标书代写
- 业务拓展：附近项目、供求市场、行业展会、AI推荐
- 数据服务：投标风险评估、投标信用报告、中标动态、政府采购中心

## 开发说明

### 样式规范

- 使用 rpx 作为尺寸单位，适配不同屏幕
